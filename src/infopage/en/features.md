# Features

## 🔐 Privacy Features

- **Zero Tracking** - No cookies required, no search logs stored
- **Tracker Blocking** - Blocks 500+ known trackers
- **URL Cleaning** - Removes UTM parameters and tracking codes
- **Anonymous Proxy** - Optional IP hiding
- **Referrer Hiding** - Websites won't know your traffic source
- **Encrypted Search** - Optional encrypted queries

## 🏆 Kagi-Style Ranking

- **Domain Pinning** - Pin favorite websites to top
- **Domain Boosting** - Prioritize trusted domains
- **Domain Blocking** - Block unwanted domains
- **Quality Scores** - Visual quality indicators
- **Trust Badges** - See trustworthy sources
- **Privacy Ranking** - Boosts privacy-respecting sites

## ⚡ Search Features

- **Calculator** - `calc 2+2`
- **Unit Conversion** - `100 km to miles`
- **Time Zones** - `time in Tokyo`
- **Search Shortcuts** - `!g`, `!w`, `!yt`
- **Keyboard Navigation** - `j/k` to navigate
